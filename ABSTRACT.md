ABSTRACT
In the modern educational landscape, students face significant decision-making challenges in study management due to increasing curriculum complexity, time constraints, inconsistent habits, and the need for effective productivity tracking. StudyBuddy aims to empower students with an accessible and interactive platform for organizing study schedules, monitoring progress in real-time, and building sustainable learning habits. Built on the React framework with Vite as the build tool, the application offers a responsive, user-friendly interface for seamless navigation and interaction, enhanced by data visualization using Recharts library featuring bar charts for daily study hours, line charts for weekly progress, and pie charts for subject-wise distribution. Users can manage personalized timetables (class schedules and study sessions across Monday–Sunday with summary statistics), utilize a Pomodoro-based study timer (25-minute focus sessions, 5-minute short breaks, 15-minute long breaks after four cycles, with subject selection and automatic session saving), access dashboard overviews (active reminders, total study hours, completion rates, current/best study streaks, weekly goals with progress bars and editable targets), handle profile management (student name/ID/course/year, achievements like tasks completed/average grade), and customize settings (dark mode, notifications, study reminders). The system integrates Firebase suite for backend capabilities: Authentication for secure email/password login/registration/password reset, Cloud Firestore for real-time synchronization of data collections (users, timetables, study sessions, tasks, reminders, settings), Cloud Messaging for web/mobile push notifications, and Analytics for behavior insights. Complementing cloud storage, localStorage enables offline functionality with dedicated keys (studybuddy_user, studybuddy_sessions, studybuddy_timetable, studybuddy_tasks, studybuddy_reminders, studybuddy_settings, studybuddy_active_session), ensuring persistence during connectivity issues. Progressive Web App (PWA) features via vite-plugin-pwa provide installability, service workers for caching, and native-like mobile experience, while Capacitor framework facilitates Android native builds with local/push notifications, exact alarm scheduling, and boot receiver persistence. Educational resources display achievement insights and streak motivators, with extensibility for community forums. The platform aids academic research through exportable Firestore data and visualized analytics for performance analysis and reporting.
Keywords: StudyBuddy, React, Vite, Firebase, Pomodoro, Analytics, PWA, Notifications, Timetable, Productivity
